gyst by mattscilipoti

A suite of helpers to aid in speeding up your tests.

A suite of helpers to aid in speeding up your tests; including scripts, existing libraries, and patches. Inspired by Nick Gauthier's ["Grease-Your-Suite" presentation(s)][gys_showoff]. Faster tests? Good.

Dependencies

No runtime dependencies (except for the libraries you choose to patch).

Install

This will be released to rubygems.org once we have a little more functionality. **NOTE:** in order to use the bash scripts, you must use the `--no-wrapper` argument. gem install pkg/gyst-0.0.1.gem --no-wrappers
